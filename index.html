About Us

samRad Health Technologies is a cancer research and technology company dedicated to developing intelligent, automated tools to assist in tumor detection and quantification from radiological imaging. We build open, reproducible AI pipelines that accelerate cancer diagnostics and research.

Our Services

AI Tumor Detection

Real-time risk assessment from DICOM images using trained models.

Clinical Pipeline Tooling

CLI-based tools for research labs to process patient datasets at scale.

Data Integration

Support for CSV metadata, structured clinical tagging, and research export.

Developer Information

This project uses Python and pandas, and provides a command-line interface for inference and CSV result logging. To run:

python main.py

Make sure to include a `pipeline_runner.py` module implementing the model logic.
